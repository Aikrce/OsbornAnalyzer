# 学术风格SVG绘制指南

## 用途
绘制AI大模型中的Transformer网络结构图（SVG）

## 核心要求

### 布局规范
- 元素不重叠，避免内容过于拥挤
- 保持清晰的层次结构和视觉层次
- 确保各组件间有适当的间距

### 内容要求
- 添加小型公式来解释关键计算
- 使用精确的专业术语
- 包含必要的技术细节和参数说明

### 字体规范
- **数学公式：** 使用公式字体
- **英文文本：** 使用 Times New Roman 字体
- **中文文本：** 使用宋体

### 视觉风格
- 注意采用业界标准的学术风格
- 绘图配色参考进行绘制
- 保持专业、严谨的视觉效果

## 质量检查清单

- [ ] 所有元素无重叠
- [ ] 内容布局合理，无拥挤感
- [ ] 包含关键计算公式
- [ ] 使用准确的专业术语
- [ ] 字体使用符合规范
- [ ] 整体风格符合学术标准
- [ ] 配色方案专业和谐

---

*本指南确保生成的SVG图表符合学术论文的出版标准。*
